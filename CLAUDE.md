## 프로젝트 개요

**Gametopia**은 AI 기반 인디 게임 마케팅 어시스턴트와 스트리머 매칭 플랫폼을 결합한 서비스입니다.

- **비전**: 좋은 게임이 묻히지 않는 세상을 만든다
- **주 타겟**: 1인 ~ 소규모 인디 게임 개발자, 스트리머/유튜버
- **핵심 문제**: 시장 포화로 인한 발견 가능성(Discoverability) 위기, 마케팅 비용 급등, 퍼블리셔 유무에 따른 생존 격차

## 아키텍처 개요

프로젝트는 두 개의 트랙으로 구성됩니다:

| 트랙    | 이름                 | 대상             | 핵심 기능                                       |
| ------- | -------------------- | ---------------- | ----------------------------------------------- |
| TRACK-A | AI 마케팅 어시스턴트 | 인디 게임 개발자 | 스팀 게임 분석, 벤치마킹, 맞춤 마케팅 전략 추천 |
| TRACK-B | 스트리머 매칭 플랫폼 | 스트리머/유튜버  | 프롬프트 기반 적합 게임 추천                    |

### 핵심 기능 (FEAT-ID)

- **FEAT-01**: AI 게임 분석 & 마케팅 추천 (TRACK-A 핵심)
- **FEAT-02**: 스팀 연동 자동 분석 (TRACK-A 지원)
- **FEAT-03**: 성공 사례 DB (TRACK-A, TRACK-B 공유)
- **FEAT-04**: 스트리머 매칭 플랫폼 (TRACK-B 핵심)

### 트랙 간 연결 구조

개발자가 TRACK-A를 통해 게임을 등록·분석하면, 해당 게임은 TRACK-B의 스트리머 매칭 풀에 자동 포함됩니다. FEAT-03(성공 사례 DB)이 두 트랙을 연결하는 중추 역할을 합니다.

## 기술 스택 (예상)

| 영역         | 기술                               | 관련 기능        |
| ------------ | ---------------------------------- | ---------------- |
| 데이터 수집  | Steam Web API, Store 페이지 크롤링 | FEAT-02          |
| AI 분석      | LLM 기반 분석 엔진                 | FEAT-01, FEAT-04 |
| 성공 사례 DB | 벡터 DB (유사도 검색) + 관계형 DB  | FEAT-03          |
| 프론트엔드   | 웹 기반 (반응형)                   | 전체             |
| 백엔드       | REST API 서버                      | 전체             |
