# 게임 피치 작성

이 스킬은 게임을 짧고 매력적으로 소개하는 피치 문구를 작성합니다.

## 사용 방법

```
/game-pitch <게임_설명>
```

## 예시

```
/game-pitch 2인 협동 공포 게임, 폐쇄된 병원 탈출, 퍼즐과 숨바꼭질
/game-pitch 우리 게임은 중세 판타지 턴제 전략 게임입니다
/game-pitch
```

## 출력 형식

- **원라이너 (One-liner)**: 한 문장으로 요약
- **짧은 피치 (Short Pitch)**: 2~3문장 설명
- **긴 피치 (Long Pitch)**: 1단락 설명
- **스팀 페이지용**: 스팀 스토어에 적합한 설명
- **프레스 키트용**: 언론/미디어 전용 설명

## 프롬프트

사용자가 게임 피치 문구를 작성하고 싶어합니다. 다음 단계를 따르세요:

1. 사용자가 게임을 설명했는지 확인하세요. 충분한 정보가 없다면 다음을 묻습니다:
   - 장르는 무엇인가?
   - 핵심 게임플레이는?
   - 타겟 오디언스는 누구인가?
   - 특별한 점(USP)은 무엇인가?

2. 다음 형식으로 피치를 작성하세요:

### 원라이너
```
[장르] 게임에서 [플레이어가 하는 일]을 경험하세요.
```

### 짧은 피치
```
[게임명]은 [장르] 게임으로, [핵심 게임플레이]를 특징으로 합니다. [USP]로 게이머들에게 새로운 경험을 제공합니다.
```

### 긴 피치
```
[게임명]에 오십시오. 여러분은 [플레이어 역할]로서, [주요 목표]를 달성해야 합니다. [핵심 메커닉]과 [특별 요소]가 결합된 이 게임은 [타겟 오디언스]에게 완벽한 선택이 될 것입니다.
```

### 스팀 스토어용
- 짧고 매력적인 문단
- 주요 특징 (Bullet points)
- 스크린샷/트레일러와 어울리는 톤

### 프레스 키트용
- 전문적이고 정보 중심의 톤
- 기자/미디어를 위한 핵심 정보 강조

3. 각 피치 버전은 서로 다른 목적과 길이를 가지도록 작성하세요.

## 참고

이 스킬은 스팀 스토어 페이지, 프레스 키트, 트레일러 등 다양한 마케팅 자료 작성을 지원합니다.
