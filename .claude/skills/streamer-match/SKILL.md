# 스트리머 게임 매칭

이 스킬은 스트리머나 유튜버의 콘텐츠 스타일을 기반으로 적합한 인디 게임을 추천합니다.

## 사용 방법

```
/streamer-match <콘텐츠_프롬프트>
```

## 예시

```
/streamer-match 2인 협동 공포 게임, 1시간 내 클리어 가능
/streamer-match 친구들과 즐기는 파티 게임, 이해하기 쉽고 반응이 좋은
/streamer-match 심리 스릴러, 스토리 중심, 5시간 정도
/streamer-match
```

## 추천 기준

- **콘텐츠 길이**: 에피소드당 플레이 시간
- **협동 여부**: 솔로/협동/멀티플레이
- **장르**: 호러, 액션, 퍼즐, RPG 등
- **난이도**: 초보자 친화적 / 하드코어
- **시각적 요소**: 반응을 이끌어내는 요소

## 프롬프트

사용자가 자신의 콘텐츠 스타일에 맞는 게임을 찾고 싶어합니다. 다음 단계를 따르세요:

1. 사용자가 원하는 콘텐츠 스타일을 프롬프트로 제공했는지 확인하세요. 제공하지 않았다면 다음을 묻습니다:
   - 몇 명이 플레이하는가? (솔로/협동/파티)
   - 에피소드당 원하는 플레이 시간은?
   - 어떤 장르를 선호하는가?
   - 시청자 반응이 중요한가? (액션, 스토리, 스폰서샵 등)

2. 프롬프트를 분석하여 다음 조건에 맞는 게임을 추천하세요:
   - 인디 게임 (스팀 등에 등록된)
   - 시청자 반응을 이끌어낼 수 있는 요소 보유
   - 요청된 플레이 시간과 일치
   - 콘텐츠 제작에 적합한 난이도

3. 추천 형식:
   - 게임명과 스팀 링크
   - 추천 이유 (콘텐츠와의 적합성)
   - 주요 특징 (장르, 플레이 타임, 난이도)
   - 관전 포인트 (시청자 반응을 이끌 요소)
   - 주의사항 (있는 경우)

4. 3~5개의 게임을 추천하고, 각각에 대해 스팀 링크를 포함하세요.
