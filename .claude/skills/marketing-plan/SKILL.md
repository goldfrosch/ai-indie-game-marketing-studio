# 마케팅 전략 추천

이 스킬은 게임에 대한 맞춤형 마케팅 전략을 제공합니다.

## 사용 방법

```
/marketing-plan <게임_설명_또는_URL>
```

## 예시

```
/marketing-plan 우리 게임은 2인 협동 공포 게임으로, 플레이어가 폐쇄된 병원을 탈출해야 합니다
/marketing-plan https://store.steampowered.com/app/xxxxx/
/marketing-plan
```

## 출력 항목

- **벤치마킹**: 유사 장르 성공 게임의 마케팅 전략 분석
- **채널 추천**: 현재 게임에 적합한 마케팅 채널 (SNS, 커뮤니티, 스트리머, 할인 이벤트 등)
- **페이지 개선**: 태그 최적화, 설명문 개선, 스크린샷 구성 제안
- **액션 플랜**: 출시 후 타임라인별 마케팅 액션 플랜

## 프롬프트

사용자가 게임에 대한 마케팅 전략을 얻고 싶어합니다. 다음 단계를 따르세요:

1. 사용자가 게임을 설명했는지, 또는 스팀 URL을 제공했는지 확인하세요. 충분한 정보가 없다면 필요한 정보를 묻습니다.

2. 게임의 특성을 파악하세요:
   - 장르, 타겟 오디언스, USP(Unique Selling Point)
   - 현재 마케팅 상황 (출시 전/후, 예산 범위 등)

3. 다음 카테고리로 마케팅 전략을 제안하세요:
   - **벤치마킹 대상**: 유사 장르/스타일의 성공한 인디 게임 3~5개
   - **채널 전략**: 효과적인 마케팅 채널별 우선순위
   - **스팀 페이지 최적화**: 태그, 스크린샷, 트레일러 개선 제안
   - **스트리머/유튜버 협업**: 접근 방법과 타겟 크리에이터 유형
   - **커뮤니티 관리**: Discord, Reddit, 소셜 미디어 전략
   - **타임라인**: 출시 전/후 주별 액션 아이템

4. 각 추천은 구체적이고 실행 가능하게 작성하세요.
