# 스팀 게임 분석

이 스킬은 스팀 게임의 URL 또는 앱 ID를 입력받아 게임을 분석하고 마케팅 관련 통찰을 제공합니다.

## 사용 방법

```
/steam-analyze <스팀_게임_URL_또는_앱_ID>
```

## 예시

```
/steam-analyze https://store.steampowered.com/app/1245620/ELDEN_RING/
/steam-analyze 1245620
/steam-analyze
```

## 분석 항목

- **기본 정보**: 장르, 태그, 가격대, 출시일
- **페이지 품질**: 스크린샷, 트레일러, 설명문 구성 및 품질 평가
- **유저 반응**: 리뷰 키워드 분석 (긍정/부정 요인 추출)
- **시장 포지션**: 경쟁 게임 대비 포지셔닝

## 프롬프트

사용자가 스팀 게임을 분석하고 싶어합니다. 다음 단계를 따르세요:

1. 사용자가 스팀 URL 또는 앱 ID를 제공했는지 확인하세요. 제공하지 않았다면 묻습니다.

2. 제공된 게임에 대해 다음을 분석하세요:
   - 게임의 기본 정보 (이름, 장르, 태그, 가격, 출시일, 개발사/퍼블리셔)
   - 스팀 스토어 페이지 구성 (스크린샷, 트레일러, 설명문 등)
   - 유저 리뷰 패턴 분석 (긍정/부정 요인)
   - 유사 장르 성공 게임과의 비교

3. 마케팅 관점에서 인사이트를 제공하세요:
   - 현재 페이지의 강점과 약점
   - 개선이 필요한 부분
   - 타겟 오디언스 분석

4. WebFetch 또는 웹 리더 도구를 사용하여 스팀 스토어 페이지의 실제 내용을 분석하세요.
