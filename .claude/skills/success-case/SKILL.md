# 성공 사례 분석

이 스킬은 마케팅으로 성공한 인디 게임 사례를 검색하고 그들의 전략을 분석합니다.

## 사용 방법

```
/success-case <검색_조건>
```

## 예시

```
/success-case 2D 픽셀아트 호러 게임
/success-case 로그라이크 게임, 2024년 이후 출시
/success-case 협동 게임, 예산 1만달러 이하
/success-case
```

## 검색 조건

- **장르**: 호러, 액션, RPG, 퍼즐, 시뮬레이션 등
- **비주얼 스타일**: 픽셀아트, 3D, 핸드드로운 등
- **출시 시기**: 특정 연도 범위
- **마케팅 예산**: 저예산/중예산/고예산
- **플랫폼**: 스팀, 콘솔, 모바일 등

## 프롬프트

사용자가 성공한 인디 게임 사례를 찾고 싶어합니다. 다음 단계를 따르세요:

1. 사용자가 검색 조건을 제공했는지 확인하세요. 제공하지 않았다면 일반적인 성공 사례를 검색하거나, 구체적인 조건을 묻습니다.

2. WebSearch를 사용하여 해당 조건에 맞는 성공 사례를 검색하세요.

3. 각 성공 사례에 대해 다음을 분석하세요:
   - **게임 개요**: 이름, 출시일, 장르, 개발사
   - **성공 지표**: 판매량, 수익, 동시 접속자 수 등
   - **마케팅 전략**:
     - 주요 마케팅 채널
     - 바이럴 요인 (트레일러, 데모, 이벤트 등)
     - 스트리머/유튜버 협업 사례
     - 커뮤니티 구축 방법
   - **핵심 요인**: 성공의 주원인

4. 분석 결과를 벤치마킹 가능한 형식으로 정리하세요:
   - 적용 가능한 전략 요약
   - 유사 장르 게임을 위한 팁
   - 피해야 할 실수 (있는 경우)

5. 2~4개의 성공 사례를 상세히 분석하세요.
